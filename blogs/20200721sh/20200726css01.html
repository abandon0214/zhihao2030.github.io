            # 1.Css文本属性

            1.color: 文字颜色;

            2.text-align:center; 文字水平位置- left(默认值)/ center/ right

            ​ text-align:center能让什么水平居中: 能让标签中的文本,行内元素,行内块元素水平居中

            ​ 从设置的角度,

            ​ 如果希望文本和行内元素和行内块元素水平居中,需要给父元素设置text-align:center

            3.font-size:字号px; 字号 需要写单位,chrome的默认字号是16px

            4.font-family:"字体"; chrome默认是微软雅黑,当时一个词组或者汉字时需要加双引号

            5.font-weight:blod； 加粗 400 normal 正常(默认值) 700 bold 加粗

            6.text-indent:2em; 首行缩进：

            ​ 单位em：一个em等于当前标签一个字号大小

            # 2.显示模式

            盒子的基本三属性：宽度，高度，背景色；width，height，background

            # 2.1.块级显示模式:

            ​ 独占一行,设置宽高起作用,在不设置宽度时,宽度和父元素一样

            # 2.1.1 块元素:

            ​ html,body,div,h1-h6,p,ul,ol,li,dl,dt,dd,hr,form

            ​ 块元素的默认宽度:块元素在不设置宽度时,和父元素一样宽

            ​

            # 2.2.行内显示模式:

            ​ 一行存在多个,设置宽高不起作用,宽高由内容决定,文本沿着基线对齐,行内显示模式的元素是用来修饰文本的

            # 2.2.1 行内元素: span,a,b,strong,i,em,u,ins,s,del

            ​

            # 2.3.行内块显示模式: 一行存在多个,设置宽高起作用的,和文本的基线对齐

            # 2.3.1.行内块元素:

            # img,昨天学习表单标签

            # 2.4.显示模式转换：

            ​ 1.其他显示模式转换成行内块显示模式

            2.其他显示模式转换成块级显示模式

            # 3.选择器

            ​ 基础选择器: 标签选择器 类选择器 id选择器

            ​ 复合选择器: 后代选择器

            3.1.类选择器

            ​ 1.定义类选择器: 以 . 开头+类名

            ​ 2.调用类选择器: 用标签中的class属性调用类名

            ​ 3.命名规范:

            ​ 1.不能用数字开头,可以是字母,下划线开头,加数字+中划线+下划线

            ​ 2.推荐使用驼峰命名法: 多个单词组成名称,第一个单词首字母小写,后面单词首字母大写

            1. 多类名调用：类名可以调用多个,用空格隔开

            # 3.2.ID选择器

            ​ 1.定义id选择器:以#开头+id名称

            \2. 调用id选择器: id属性等于id名称

            \3. 命名规范和类名一样

            ​

            ​ 类名相当于是人的名字,可以重复使用

            ​ id名相当于是身份证,身份证是唯一的

            # 3.3.后代选择器

            ​ 选择器空格后代(子子孙孙)选择器

            .web h2 h2{
                color:yellowgreen;
            }
            .web .teacher{
                color:red;

            # 4.Css的层叠性

            1.设置css属性时,不同的属性都可以实现（如分别设置字体，颜色，字体大小等不同属性）

            2.相同属性,当权重相同时,后定义的会层叠先定义的,当权重同时,谁的权重高实现谁的

            \3. 权重:

            标签选择器 < 类选择器 < id选择器 < 行内样式 < !important

            ​ 1 10 100 1000 无穷
